# IMwalleT Proxy Server

Proxy server for BudgetIQ to route IMwalleT recharge API calls through a server that can reach the IMwalleT partner API.

## One-Click Deploy

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/anirudhatalmale6-alt/imwallet-proxy)

## Environment Variables

- `PROXY_SECRET` - Authentication secret (default: `biq_imw_proxy_2026`)

## After Deploy

Copy the deployed URL (e.g., `https://imwallet-proxy.onrender.com`) and update BudgetIQ config.
